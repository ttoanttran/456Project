# 456Project

This project implements a Convolutional Neural Network (CNN) to classify images into various categories using the **Natural Images Dataset**.

## Dataset

The dataset used for this project is the **Natural Images Dataset**, available on Kaggle:  
[**Natural Images Dataset**](https://www.kaggle.com/datasets/prasunroy/natural-images)

The dataset contains labeled images of the following classes:  
- Airplane  
- Car  
- Cat  
- Dog  
- Flower  
- Fruit  
- Motorbike  
- Human  

## Requirements

- Python 
- TensorFlow/Keras  
- Matplotlib  
- Kaggle API (optional for dataset download)  

## Instructions

1. Download the dataset from Kaggle: [Natural Images Dataset](https://www.kaggle.com/datasets/prasunroy/natural-images).  
2. Clone this repository.  
3. Place the dataset in the appropriate directory (`natural-images/`).
